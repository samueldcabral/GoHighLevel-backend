## Instalação

Caso queira replicar este projeto, a instalação é simples.

```bash
git clone https://github.com/samueldcabral/React-Typescript-Boilerplate-and-Readme.git
cd React-Typescript-Boilerplate-and-Readme/
npm install
npm start
```
